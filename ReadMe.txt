Game Controls:
LEFT ARROW & A 	= Move left
RIGHT ARROW & D = Move Right
R		= Restart current level
M		= Menu
ESC		= Close

*in main menu type "ASSIST" for the secret level 
================================================================================================
!!Dev only controls!!
L		= Load next level
K		= Load previous level
================================================================================================
for creating a new level from unity project:

-In any level we need to assign the Player for all the NumberSquare entities and for the Liquid
-To lock a single NumberSquare we need to add a coin and add it to the coin's Inspector
